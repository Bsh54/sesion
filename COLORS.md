# Sesión — Colors

The "yellow" accent is actually an **electric lime**.

## Signature accent
| Name | HEX | RGB | Usage |
|------|-----|-----|-------|
| **Lime** | `#C8FF3D` | `200, 255, 61` | THE accent — highlights, price, active states, badges, primary "positive" buttons |

## Full palette
| Token | HEX | Usage |
|-------|-----|-------|
| Background (cream) | `#FAF7F0` | page background |
| Surface | `#FFFFFF` | cards, sheets |
| Ink | `#14161B` | text, primary buttons |
| Ink soft | `#565A66` | secondary text |
| **Lime (accent)** | `#C8FF3D` | accent / highlight |
| Coral | `#FF5A3C` | urgency ("spots left"), live |
| Success | `#16A34A` | payment ok, check-in |
| Border | `#E9E4D8` | separators |
| Muted | `#F1EEE6` | inert fills, skeletons |
| Destructive | `#DC2626` | cancel / errors |

> Tokens are defined in `tailwind.config.js` and documented in full in `DESIGN.md`.
